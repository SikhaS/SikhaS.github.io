MPICV

Magnetic Partical Inspection using Computer Vision. (crack detection), NDT, CV Quality is a key role in any inspection process. Computer-vision based magnetic particle inspection (MPICV) is accurate, precise, reliable and is designed for the inspection of ferrous components up to 600 mm in length. This self-contained system with the camera sensor, edge device, UV light source is a low-cost unit that detects and estimate the length of surface cracks using the wet continuous method of MPI. This magnetic particle inspection with CV would be a reinforcement learning-based system that will continuously learn and make itself accurate to detect cracks and length of cracks with the help of AI/ML algorithms with precision and accuracy.
PufferFish ventilator (Stanford University)

PufferFish is a rapidly manufacturable open-source full-featured ventilator. Pufferfish is an open-source, full-featured ICU ventilator. It is the result of a communal effort to design a low-cost solution, that can be manufactured rapidly and is clinically relevant for COVID-19 patients. With a peak flow rate above 300 l/min, Pufferfish can be used to provide both invasive and noninvasive ventilation as well as high flow oxygen therapy.
